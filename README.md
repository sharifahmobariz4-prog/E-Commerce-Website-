# E-Commerce-Website-
# ✨ Sharifah Shop

Welcome to Sharifah Shop — a modern Hazara fashion e-commerce website inspired by cultural beauty, traditional style, and modern online shopping experiences.

This project was built using HTML, CSS, Bootstrap, and JavaScript with a responsive and elegant design.

---

## 🌸 Features

- 🛍 Modern Online Shop
- 👗 Hazara Fashion Collection
- 💎 Jewelry & Accessories
- 👡 Traditional Shoes Collection
- 🛒 Shopping Cart System
- 💳 Checkout Page
- 👤 Login Page
- 📱 Fully Responsive Design
- 🌙 Dark Mode Support
- 🔍 Product Search & Categories

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

---

## 🎨 Project Goals

This project was created to combine:
- traditional Hazara fashion
- modern web design
- responsive e-commerce experience

while improving frontend development skills and UI design creativity.
